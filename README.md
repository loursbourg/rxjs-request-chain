# Chain sequential Http requests with RxJS

_By [Kadir Damene](https://github.com/loursbourg)_

This example demonstrates how to chain multiple Http requests using RxJS and axios.

### Operators Used

- catchError
- concatMap
- delay
- map
- mergeMap
- take (optional)
